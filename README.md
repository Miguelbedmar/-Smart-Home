# -Smart-Home


#OBJETIVO.
EL objetivo del proyecto es crear una landing page tipo Dashboard  de control para una Smart Home única en donde se apliquen todos los conocimientos vistos en la asignatura de lenguaje de marca.Utilizando cuadrículas que utilicen diferentes widgets temperatura,luces,cámaras o consumo enérgitico entre muchas más.

#REQUISITOS DEL PROYECTO O PREMISAS.


1.Restricción: No se permite el uso de píxeles (px) para font-size, padding, margin o anchos de contenedor.

2.Obligación: Deben usar rem para tipografía y espaciados, y em para componentes contenidos. Los anchos deben ser en %, vw/vh o fracciones de grid (fr)

3.Restricción: No se puede definir cada celda manualmente.

4.Obligación: El dashboard debe usar grid-template-columns: repeat(auto-fill, minmax(300px, 1fr)) en al menos una sección.

5.Obligación: Deben crear un sistema de capas de abstracción para las variables:

5.1Ejemplo Globales: --color-blue-500: #2563eb;

5.2Ejemplo de variables: --accent-color: var(--color-blue-500);

6.Uso en Componentes:border: 1px solid var(--accent-color);

7.El proyecto debe construirse bajo la filosofía Mobile-First. El CSS base debe ser la versión móvil, y las Media Queries deben usar exclusivamente min-width.
